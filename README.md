# ESP32 Car Control (Wi-Fi Web Interface)

Web-управление машинкой на **ESP32** через Wi-Fi без роутера.  
ESP32 поднимает собственную точку доступа и отдает веб-страницу с рулём и кнопками движения.

---

## Features

- Wi-Fi Access Point (без внешнего роутера)
- Веб-интерфейс (работает с телефона и ПК)
- Руль слева с плавным управлением сервоприводом
- Автовозврат руля в центр
- Кнопки движения:
  - ▲ Forward
  - ▼ Backward
- Управление через WebSocket

---

## Hardware Requirements

- ESP32 (любой dev board)
- Сервопривод (SG90 / MG90 / аналог)
- Драйвер моторов (L298N, L9110, IBT и др.)
- Двигатель(и)
- Отдельное питание для моторов (рекомендуется)

---

## Minimal Wiring

| Function       | ESP32 Pin |
|---------------|----------|
| Servo signal  | GPIO 13  |
| Servo VCC     | 5V       |
| Servo GND     | GND      |

> ⚠️ Если используется отдельное питание для сервопривода или моторов,  
> **GND должен быть общим с ESP32**.

---

## How It Works

1. ESP32 запускается в режиме **Access Point**
2. Поднимается Web-сервер (порт `80`)
3. Управление передается через **WebSocket** (порт `81`)

### Commands

- Steering angle: `-45 … +45`
- `F` — движение вперед
- `B` — движение назад
